键帽

材质

ABS

  1. 优缺点

    • 它的优点是爽脆、轻盈、耐热、有韧性、且表面光滑、触感很细腻

    • 它的缺点是不耐磨且易打油，打油指的是表面容易被磨的很光滑

  2. 价位——10元～1000元不等

PBT

  • 它的优点是硬度高、不易磨损、不易打油
  • 它的缺点是易变形、色泽没有ABS键帽鲜艳

POM

  • 密度大、不打油、自润滑、顺畅、丝滑
  • 颜色单一、制作难度大、制作成本很高

键帽字符工艺

镭雕

采用激光雕刻字符

  • 成本低、表面字符不平整、长期使用字符会被磨掉

二色工艺（二色成型、双色注塑）

  • 制作成本较高、基本不会被磨掉

热升华工艺

在高温下，将固体树脂油墨升华为气态模式，然后让油墨渗透到键帽中，最后冷却凝华，形成完整键帽。

  • 只有PBT材质才能够进行热升华
  • 能让字符更具光泽，且不易被磨掉
  • 热升华的大键位键帽容易出现弯曲变形的情况
  • 成本适中，适合新手入门

键帽的高度

原厂高度

原厂指的是Cherry，使用较为舒适，能够还原各种轴的手感

OEM

代工厂 ，比原厂高度高一层

SA高球帽

侧边有较大的弧度、观感很好、很圆润、整体颜值很高

高度较高，且弧度牺牲打字面积，一般需要用到手托

DSA键帽

所有键帽都处于同一个高度
